BEM architecture (block, element, modifier)
  • Block: the parent
    • You can use digits, letters and hyphen
    • For example: .card
  • Element: the Child
    • You can use digits, letters, hyphen and underscore (to know that is related to the child)
    • For example: .card__title
  • Modifier
    • --modifier
    • For example: .card__title--larger
Box-sizing: border-box
Imagine the code: {width:150px; Padding:10px} If you use box-sizing:border-box the total width will be 150px, if you don’t use it it will be 170px
